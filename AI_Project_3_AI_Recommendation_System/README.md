# AI Recommendation System

## Overview

This project implements a Content-Based Recommendation System using TF-IDF Vectorization and Cosine Similarity.

The system analyzes user interests and recommends the most relevant courses by comparing user preferences with course attributes.

Developed as part of the DecodeLabs Artificial Intelligence Internship Program.

## Features

* User Preference Collection
* TF-IDF Feature Extraction
* Cosine Similarity Matching
* Top-N Recommendation Generation
* Similarity Score Ranking
