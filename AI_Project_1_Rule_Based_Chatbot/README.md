# Rule-Based AI Chatbot

## Overview

This project is a Rule-Based Artificial Intelligence Chatbot developed using Python. The chatbot simulates human conversation through predefined logical rules and decision-making algorithms. It demonstrates fundamental AI concepts such as control flow, pattern matching, user interaction management, and response generation.

Developed as part of the DecodeLabs Artificial Intelligence Internship Program.

## System Workflow

1. User enters a query.
2. Chatbot analyzes input.
3. Appropriate rule is matched.
4. Response is generated.
5. Conversation history is updated.
6. Session continues until exit command.
