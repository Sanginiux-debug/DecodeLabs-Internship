# Data Classification Using AI

## Overview

This project demonstrates supervised machine learning using the Iris Flower Dataset. A Decision Tree Classifier is trained to identify flower species based on physical measurements.

Developed as part of the DecodeLabs Artificial Intelligence Internship Program.

---

## Features

* Dataset Loading
* Data Analysis
* Train-Test Split
* Model Training
* Decision Tree Classification
* Accuracy Evaluation
* Confusion Matrix
* Classification Report
* Real-Time Prediction System

---

## Technologies Used

* Python
* Pandas
* Scikit-Learn

---

## Machine Learning Workflow

Dataset → Preprocessing → Training → Testing → Evaluation → Prediction

---

## Results

The model achieves high classification accuracy on unseen test data and can predict flower species from user-provided measurements.
